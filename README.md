# Thermal Model Analysis

This project presents a computational and experimental analysis of heat transfer between water and the surrounding environment using different insulation conditions.

## 📊 Overview

The study evaluates how temperature evolves over time in three different containers (A, B, and C) with varying thermal insulation properties. Experimental data was collected and analyzed using multiple regression models.

## ⚙️ Methods

The following models were implemented and compared:

- Linear regression  
- Quadratic regression  
- Exponential regression  

Additionally, error analysis was performed using:

- Relative error (%)  
- Standard error of estimate  
- Coefficient of determination (R²)  
- Correlation coefficient (r)

## 📈 Results

- The quadratic model provided the best statistical fit.
- The exponential model did not perform well due to the absence of an asymptotic term representing ambient temperature.
- Heat transfer behavior showed a decreasing rate over time, consistent with thermodynamic principles.

## 🧠 Engineering Applications

This analysis is relevant for:

- Biomedical Engineering (thermal regulation, sample preservation)
- Environmental Engineering (temperature dynamics in ecosystems)
- Chemical Engineering (thermal control in reactors)
